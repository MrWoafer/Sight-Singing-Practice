# Sight-Singing-Practice

A program to help practice sheet music sight-singing through simple games. Includes a small custom music theory library. Renders sheet music.

Written in C# with Unity.

# Music Theory Library
* Notes
  * Note names (e.g. C)
  * Notes with pitch (e.g. C4)
  * Notes with pitch + duration (e.g. C4 crotchet)
* Intervals (e.g. major 3rd)
* Durations (e.g. crotchet)
* Rests (e.g. crotchet rest)
* Keys (e.g. C major)
  * Scale degrees (e.g. b3)
* Time signatures (e.g. 4/4)

# Learning Games
* Name key signature
* Name note
* Name scale degree
* Plot scale degree
* Name interval
* Sing interval
* Sing melody
